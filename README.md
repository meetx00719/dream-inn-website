# Dream Inn Website

Premium React + Firebase-ready hotel/motel website for Dream Inn.

## Setup

```bash
npm install
npm run dev
```

## Firebase

1. Copy `.env.example` to `.env`
2. Add your Firebase project values
3. Firestore booking inquiries are saved to `bookingInquiries`

## Booking Engine

When your channel manager gives you a booking URL or widget script, replace the booking placeholder links/buttons in the booking section.
